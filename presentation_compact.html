スライド6：Cursorでの開発フロー

Step 1：Composerで指示

Cursor Composer（Cmd+I）で入力：

「営業日報テキストから顧客名、商談内容、
見込み金額を抽出してExcelに出力する
Pythonプログラムを作成してください。
OpenAI APIのgpt-4o-miniを使用。」

Step 2：AIがコード生成

→ main.py（メインプログラム）
→ requirements.txt（必要なライブラリ）
→ サンプルデータ
を自動生成（約2分）

Step 3：実行

$ python main.py

→ Excelファイルが出力される

Step 4：改善

Cursor Chat（Cmd+L）で：
「見込み金額も抽出して」
「確度をA/B/Cで判定して」

→ AIが即座にコード修正（約30秒）

このサイクルを繰り返すだけ！
